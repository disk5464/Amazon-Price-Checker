# Amazon-Price-Checker
This script uses SerpApi to get the current prices of differnt energy drinks on Amazon and tells you it they are currently on sale or not

It uses SerpApi to scrape the amazon data, then streamlit to display the data on a webpage. 

Currently the products are found using their Amazon ASIN codes. These are hard coded in an array at the beginning of the script. Refer to "Just ASIN Codes" to see what each code maps to.
